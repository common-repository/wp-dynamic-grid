Wordpress dynamic grid Plugin. 

== Description ==

"wp dynamic grid" can show any database table data in grid view at frontend.User can can easily show the grid at any page by using the shortcode.User can also sort the grid data at frontend. This plugin is compatible up to Wordpress Version: 3.5.0 or higher.

This is a very first release with many limitations.If any bug found please ask me for support!

Info and samples at <a href="http://ecommerce-plugin.codeinterest.com/wordpress/wp-dynamic-grid">wp dynamic grid developer's site</a>

== Installation ==

1. Upload 'wp-dynamic-grid' folder to the '/wp-content/plugins/' directory.

2. Activate 'wp-dynamic-grid' form wp plugin option in admin area.

3. Plugin will appear in the menu bar of WP Dashboard.

4. set the plugin settings.

5. Use short code([wpcustomtable]) at any page to show the grid at frontend.
